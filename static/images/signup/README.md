# Imagini – înregistrare / signup

## `backgrounds/`

Poze full-bleed sau hero pentru pagini de tip: alegere tip cont, PF, ONG, Colaborator etc.

**Static în template:** `{% static 'images/signup/backgrounds/nume-fisier.ext' %}`

## Convenții

- Nume fără spații: ex. `signup-hero-ong-01.jpg`.
- Preferă aceeași linie cromatică ca Intra (tonuri reci / neutre) dacă vrei consistență pe fluxul „cont nou”.
