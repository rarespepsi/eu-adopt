# Imagini – cont (fără login)

## `backgrounds/`

Pentru pagini legate de cont **după** sau **în afara** ecranului Intra, de exemplu:

- parolă uitată / reset
- mesaje de confirmare email (dacă au layout cu hero)
- alte ecrane „account” care nu sunt `login.html`

**Static:** `{% static 'images/account/backgrounds/nume-fisier.ext' %}`

## Notă

Intra rămâne sub `images/login/` (vezi acel README).
