# Imagini – Shop

## `backgrounds/`

Hero-uri, bannere de fundal sau atmosferă pentru pagina Shop și subpagini (dacă folosiți imagini full-bleed).

**Static:** `{% static 'images/shop/backgrounds/nume-fisier.ext' %}`

## Convenții

- Produsele propriu-zise pot sta și în alt subfolder ulterior (ex. `shop/products/`); aici e pentru fundaluri / promo vizual mare.
- Nume fără spații: ex. `shop-hero-01.webp`.
