# `static/images` – structură pe zone

| Folder | Scop |
|--------|------|
| `login/` | Intra + `backgrounds/` pentru hero full-bleed. Detalii: `login/README.md`. |
| `signup/` | Înregistrare (tipuri cont); `backgrounds/` pentru poze dedicate. |
| `account/` | Flux cont în afara login (ex. reset parolă); `backgrounds/`. |
| `common/` | Fundaluri / texturi reutilizabile între pagini. |
| `errors/` | 404, 503, mentenanță; `backgrounds/`. Detalii: `errors/README.md`. |
| `shop/` | Shop / promo vizual mare; `backgrounds/`. Detalii: `shop/README.md`. |
| `parteneri/` | Logo-uri parteneri (existent). |
| `pets/` | Conținut animale (existent). |
| `transport/` | Ilustrații transport (existent). |

Fișierele noi de tip hero: păstrează **fără spații** în nume; preferă JPG/WebP pentru fotografii mari, PNG când ai nevoie de transparență.
