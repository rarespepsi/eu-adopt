# Imagini – erori / stare specială

## `backgrounds/`

Poze sau ilustrații pentru pagini de tip 404, 503, mentenanță, „ceva nu a mers” etc.

**Static:** `{% static 'images/errors/backgrounds/nume-fisier.ext' %}`

## Convenții

- Ton calm, fără culori țipătoare; aliniat la restul site-ului (dark sau light, după template).
- Nume fără spații: ex. `error-404-hero.jpg`.
