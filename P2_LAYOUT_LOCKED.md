# P2_LAYOUT_LOCKED

**Checkpoint:** P2 (Prietenul Tău main grid) structure is finalized and frozen.

---

## Description

The P2 grid structure is locked. Do not change P2 layout in future edits. Only content inside cells may change.

---

## Locked P2 structure

| Rule | Value |
|------|--------|
| **Columns** | 4 |
| **Visible rows** | 3 (equal height) |
| **Scroll** | Internal scroll only (viewport + rest) |
| **Cell size** | Equal width and height per row/column |
| **Layout** | No layout changes allowed |

---

## Rules

- **Do not** change number of columns (must remain 4).
- **Do not** change number of visible rows (must remain 3 equal rows).
- **Do not** remove or alter internal scroll behavior.
- **Do not** change grid structure, gap, or cell sizing rules that affect layout.
- **Only** content inside cells (e.g. card content, images, text) may change.

---

*Checkpoint: P2_LAYOUT_LOCKED*
